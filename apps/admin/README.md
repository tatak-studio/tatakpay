# Tatak Pay - Admin Portal

The Admin Portal is a Next.js application designed for business owners using the Tatak Pay platform.

## Features

- **Dashboard**: Overview of business performance and recent activities.
- **Invoicing**: Create and manage invoices for customers.
- **Transactions**: View comprehensive transaction history.
- **Settings**: Configure business profile and payment settings.

## Getting Started

Run the development server:

```bash
pnpm dev
```

The application typically runs on [http://localhost:3001](http://localhost:3001).

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: React Query (TanStack Query)
