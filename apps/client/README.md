# Tatak Pay - Customer Dashboard

The Customer Dashboard is a Next.js application allowing end-customers to interact with invoices and payments.

## Features

- **View Invoices**: Access and review pending and paid invoices.
- **Make Payments**: Securely pay invoices using integrated payment gateways (Xendit).
- **Payment History**: View historical transaction records.

## Getting Started

Run the development server:

```bash
pnpm dev
```

The application typically runs on [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: React Query (TanStack Query)
